# کلاس ارتباط با وب سرویس شرکت 
یک کلاس برای ارسال و دریافت داده های شرکت های چندفروشگاهی به api شرکت پست که با اخرین ورژن اعلام شده از طرف شرکت پست (v7) هماهنگ است از طرفی دو کلاس دیگر برای ساخت فاکتور برای مرسوله و بارکد برای کد مرسوله در پکیج موجود میباشد
## نصب با کامپوزر
```shell
composer require mhmmdq/ecommerce-api
```
